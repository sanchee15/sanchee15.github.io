DL Models

Deep Learning is used to solve image

categorization and segmentation,

object detection, video description,

audio classification, game playing,

speech recognition and synthesis,

machine translation, driving cars....

Let's see what makes it possible!

Where we are now?

  • CNN

    • CNN uses convolution behind the scenes and uses the concept of parameter sharing which reduces the size of the network. This is very successful in solving image related scenarios.

  • RNN

    • RNN's use their internal memory to solve sequence related problems. It is the loops in the network that give them the capability to model long term dependencies. They are best for modelling language tasks. eg. LSTM and Attention

Where we are now?

  • Encoder Decoder

    • It is a compression algorithm which reduces the size of network. This dimensionality reduction technique ensures it efficiently learns the network representation so that decoding is easy and reconstruction of input is possible.

  • Deep Q Network

    • A model free reinforcement learning method which is used for finding optimal action selection policy. Every action leads to a reward. eg. Agents playing Atari games

Where we are headed to?

  • Deep Generative Models

    • A model for randomly generating observable data values. The model once trained gives us probablity distribution over a given set of inputs and can be used to generate more samples similar to the one in the input. eg. GAN's are used to create undistinguishable fake images.

  • Multimodal Deep Learning

    • This types of learning involves relating information from multiple sources just the way a human does. eg. Learning from audio and video together.

Where we want to be?

  • One Shot Learning
  • Meta Learning
  • Human Level Intelligence

The Ultimate Level

  • Superintelligence

    • A superintelligence is a hypothetical agent that possesses intelligence far surpassing that of the brightest and most gifted human minds.
